**TSParse** is a typescript parser written in powershell
# How to use:
Usage:
  ```tsparser.ps1 <file.ts> [ast|ssa] [-verbose]```

### Options:
 
  **ast       gen AST output (default)**
  
  **ssa       gen SSA output**
  
  **-verbose  Enable logging**
